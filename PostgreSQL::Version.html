NAME

PostgreSQL::Version - class representing PostgreSQL version numbers

SYNOPSIS

  use PostgreSQL::Version;

  my $version = PostgreSQL::Version->new($version_arg);

  # compare two versions
  my $bool = $version1 <= $version2;

  # or compare with a number
  $bool = $version < 12;

  # or with a string
  $bool = $version lt "13.1";

  # interpolate in a string
  my $stringyval = "version: $version";

  # get the major version
  my $maj = $version->major;

DESCRIPTION

PostgreSQL::Version encapsulates Postgres version numbers, providing parsing of common version formats and comparison operations.

METHODS

PostgreSQL::Version->new($version)

Create a new PostgreSQL::Version instance.

The argument can be a number like 12, or a string like '12.2' or the output of a Postgres command like `psql --version` or `pg_config --version`;

major([separator => 'char'])

Returns the major version. For versions before 10 the parts are separated by a dot unless the separator argument is given.
